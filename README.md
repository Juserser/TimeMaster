# 🕒 TimeMaster (타임마스터)

**TimeMaster**는 효율적인 시간 관리를 위해 설계된 스타일리시하고 강력한 **데스크탑 타임박싱(Time-Boxing) 위젯**입니다. 일론 머스크의 시간 관리법으로 유명한 타임박싱 기법을 아름다운 Glassmorphism UI와 함께 경험해 보세요.

---

## 🚀 v0.2.0 업데이트 주요 사항 (Release Notes)

이번 버전에서는 **플랜 상세 관리 기능**과 **생산성 강화 기능**이 대거 추가되었습니다.

- **플랜 메모 & 링크**: 타임박스 블록 또는 할 일을 클릭하면 하단 패널이 해당 항목의 전용 메모장 + 링크 관리 뷰로 전환됩니다. 메모 영역과 링크 영역의 높이를 드래그로 자유롭게 조절할 수 있습니다.
- **파일 & URL 바로가기**: 링크 섹션에 URL, 로컬 파일, 바로가기(.lnk)를 추가하고 바로 열 수 있습니다. 파일 탐색기에서 드래그 앤 드롭으로 간편하게 등록하며, 링크 이름은 더블클릭으로 수정 가능합니다.
- **타임박스 인터랙션 개편**: 블록 좌클릭 → 메모/링크 패널 열기 / 블록 우클릭 → 편집·삭제·색상 변경 컨텍스트 메뉴
- **그리드 선 숨기기**: 메뉴에서 타임박스 그리드 선 표시 여부를 토글할 수 있습니다. 블록 배경이 불투명하게 전환되어 선이 완전히 가려집니다.
- **미완료 할 일 내일로 넘기기**: 메뉴의 "미완료 할 일 내일로 넘기기"로 오늘 미완료 항목 전체를 다음 날로 일괄 이동합니다. 개별 항목은 우클릭 → "내일로 이동"으로 처리합니다.
- **타임박스 색상 개선**: 커스텀 색상 적용 시 기본 블록과 동일한 4px 좌측 테두리 디자인을 유지하며, 색상은 은은한 20% 투명도로 표시됩니다.

## 🚀 v0.1.1 업데이트 주요 사항 (Release Notes)

이번 버전에서는 데스크탑 위젯으로서의 활용성을 극대화하기 위해 **월간 캘린더 모드**와 **화면 제어 기능**이 대폭 강화되었습니다.

- **전체화면 캘린더 모드 (New)**: 상단 토글을 통해 타임박스 그리드와 월간 캘린더 뷰를 자유롭게 전환할 수 있습니다. 한 달간의 전체적인 일정 흐름을 한눈에 파악하세요.
- **클릭 무시 (Click-through) 모드**: 앱을 화면 최상단에 띄워둔 상태에서 마우스 클릭만 통과시키는 모드를 지원합니다. 배경화면처럼 쓰면서 뒤의 창을 그대로 조작할 수 있습니다.
- **최상단 고정 (Always on Top)**: 어떤 작업을 하든 타임박스가 가려지지 않도록 화면 가장 앞쪽에 고정할 수 있습니다.
- **투명도 한계 돌파 (10%)**: 기존 30%였던 최소 투명도를 **10%**까지 낮출 수 있게 되어, 배경화면과 더욱 완벽하게 조화됩니다.
- **블러(Blur) 최적화**: 낮은 투명도에서도 뒷 배경이 더 선명하게 보이도록 배경 흐림 강도를 절반(12px)으로 조정했습니다.
- **극단적 레이아웃 조절**: 중앙 조절 바의 가동 범위를 **5%~95%**로 확대하여, 한쪽 패널을 거의 숨기고 필요한 정보만 넓게 볼 수 있습니다.
- **고급 설정 메뉴 정리**: 저사양 모드, 시간선, 시간 뱃지 등 유틸리티 설정들을 '고급 설정' 그룹으로 묶어 메인 메뉴를 깔끔하게 정리했습니다.

## ✨ 핵심 기능 (Key Features)

### 1. 시각적 타임박싱 그리드 (Time-Boxing Grid)
- **드래그 앤 드롭 일정 관리**: 마우스 드래그로 일정을 자유롭게 배치하고 옮길 수 있습니다.
- **자유로운 리사이징**: 일정 블록의 상/하단을 드래그하여 5분 단위로 정밀하게 시간을 조절합니다.
- **인터벌 커스터마이징**: 그리드 간격을 5분, 15분, 30분 단위로 변경할 수 있습니다.
- **실시간 시간선 & 뱃지**: 그리드 위에 현재 시간을 알려주는 **움직이는 시간선(Timeline)**과 현재 시각 뱃지를 표시합니다.
- **드래그 스크롤 (Drag-to-Scroll)**: 그리드의 빈 공간을 잡고 드래그하여 화면을 부드럽게 상하로 스크롤할 수 있습니다.

### 2. 마스터 할 일 목록 (Master Tasks)
- **할 일 집중 관리**: 당장 시간을 배정하지 않은 할 일들을 보관하고 관리합니다.
- **간편한 배정**: 할 일 목록에서 그리드로 드래그하면 즉시 타임박스 일정이 생성됩니다.
- **순서 재정렬**: 할 일을 드래그하여 우선순위에 따라 자유롭게 순서를 바꿀 수 있습니다.
- **컬러 라벨 & 즉시 수정**: 할 일마다 고유 색상을 지정하거나 더블 클릭으로 내용을 즉시 수정할 수 있습니다.

### 3. 고도화된 캘린더 모드
- **월간 일정 대시보드**: 각 날짜 칸에 할 일들이 칩 형태로 표시되어 월간 성취도를 한눈에 확인합니다.
- **호버 프리뷰**: 날짜 위에 마우스를 올리면 그날의 전체 할 일 리스트를 툴팁으로 즉시 미리 볼 수 있습니다.
- **완벽한 데이터 연동**: 캘린더에서 날짜를 클릭하면 즉시 해당 날짜의 타임박스 화면으로 이동합니다.
- **자정 자동 갱신**: 자정이 지나면 현재 화면을 오늘 날짜로 자동 업데이트하여 연속성을 유지합니다.

### 4. 강력한 커스터마이징 & UI
- **개인화된 타이틀**: 상단 'TimeMaster' 문구를 클릭하여 나만의 위젯 이름으로 변경할 수 있습니다.
- **지능형 투명도 동기화**: 위젯 투명도 조절 시 모든 메뉴, 달력, 우클릭 메뉴의 투명도가 유기적으로 함께 조절됩니다.
- **6종 테마 프리셋**: Deep Space, White Pearl, Emerald 등 분위기에 맞는 테마를 제공합니다.
- **유연한 레이아웃**: 중앙 바와 메모장 높이 조절 바를 통해 자신에게 최적화된 화면 구성을 만듭니다.

### 5. 데스크탑 특화 기능 (Electron)
- **데스크탑 알림**: 일정이 시작될 때 시스템 알림을 통해 할 일을 놓치지 않게 도와줍니다.
- **트레이 메뉴 동기화**: 창을 닫아도 트레이에서 실행되며, 트레이 아이콘 우클릭으로 주요 설정을 즉시 변경할 수 있습니다.
- **자동 실행 (Auto-launch)**: PC 부팅 시 앱이 자동으로 시작되도록 설정 가능합니다.
- **상태 보존**: 앱의 위치, 크기, 패널 너비 등 모든 상태가 종료 후 재실행 시 그대로 유지됩니다.
- **저사양 모드**: 애니메이션과 블러 효과를 최소화하여 성능이 낮은 환경에서도 쾌적하게 작동합니다.

### 6. 데이터 안전성 & 관리
- **로컬 자동 저장**: 모든 데이터는 클라우드가 아닌 사용자 PC의 안전한 로컬 저장소에 실시간 기록됩니다.
- **백업 및 복구**: 전체 데이터를 JSON 파일로 내보내거나 가져올 수 있으며, 종료 시 자동 백업을 생성합니다.
- **데이터 마이그레이션**: 기존 브라우저 데이터를 파일 시스템 저장소로 안전하게 자동 이전합니다.

- **Frontend**: React 18, TypeScript, Vite

- **Styling**: Vanilla CSS (Custom Glassmorphism & Animations)
- **Storage**: Electron Store (JSON File Based System)

## 🚀 시작하기 (How to Use)

1. **뷰 모드 전환**: 상단 중앙의 [타임박스 / 캘린더] 버튼으로 모드를 바꿀 수 있습니다.
2. **일정 추가**: 그리드의 빈 칸을 클릭하거나 할 일을 목록에서 드래그하세요.
3. **클릭 무시 해제**: 해당 모드 중에는 앱 클릭이 안 됩니다. 화면 하단 **트레이 아이콘을 우클릭**하여 설정을 해제하세요.
4. **창 이동**: 상단의 투명한 드래그 핸들(날짜 왼쪽 영역 등)을 잡고 창을 옮길 수 있습니다.
