# 프로젝트: Elon-TimeBox (일론 머스크 타임 박싱 위젯)

본 프로젝트는 일론 머스크의 5분 단위 시간 관리 기법(Time Boxing)을 바탕화면 위젯 형태로 구현하여 사용자의 생산성을 극대화하는 것을 목표로 합니다.

## 1. 개요 (Overview)
*   **컨셉**: 바탕화면에 항상 떠 있는 '스티커 메모' 스타일의 초정밀 일정 관리기.
*   **철학**: "계획하지 않은 시간은 사라진 시간이다." 5분 단위의 블록을 통해 몰입도를 높임.
*   **타겟**: 초집중 상태(Deep Work)가 필요한 개발자, 학생, 전문직.

## 2. 주요 기능 (Core Features)

### MVP (1단계)
1.  **사용자 정의 그리드(Interval)**: 5분, 15분, 30분, 60분 중 원하는 시간 관리 단위를 선택 가능.
2.  **동적 그리드 시각화**: 선택한 단위에 맞춰 하루 일정을 칸으로 나누어 표시.
3.  **드래그 앤 드롭 블록 생성**: 마우스 드래그로 일정의 시작과 끝을 지정.
4.  **글래스모피즘(Glassmorphism) UI**: 배경이 비치는 반투명한 세련된 위젯 디자인.
5.  **현재 상태 트래킹**: 현재 시간에 해당하는 블록을 강조하고, 지난 시간은 어둡게 처리.
6.  **로컬 저장**: 위젯을 껐다 켜도 작성한 일정과 설정값이 `localStorage`를 통해 유지.

### 확장 기능 (2단계)
1.  **사운드 알림**: 5분 블록이 끝날 때마다 아주 짧고 고급스러운 효과음 송출.
2.  **통계 리포트**: 하루 중 얼마나 많은 블록을 '완료'로 표시했는지 생산성 점수 계산.
3.  **다크/라이트 모드**: 사용자의 OS 테마에 맞춘 자동 전환.
4.  **집중 모드 (Lock)**: 정해진 시간 동안 다른 앱 사용을 자제하도록 유도하는 시각적 피드백.

## 3. 기술 스택 (Technical Stack)
*   **프레임워크**: Electron (데스크톱 위젯 구현)
*   **프론트엔드**: React, TypeScript (안정적인 상태 관리)
*   **스타일링**: Vanilla CSS (고급스러운 유리 질감 및 애니메이션 최적화)
*   **빌드 도구**: Vite (빠른 개발 및 빌드 환경)

## 4. UI/UX 디자인 가이드라인
*   **테마**: 다크 모드 (SpaceX/Tesla 스타일의 네온 블루 및 화이트 폰트).
*   **효과**: 
    *   `backdrop-filter: blur(10px)`를 활용한 유리 효과.
    *   `border: 1px solid rgba(255, 255, 255, 0.1)`를 통한 은은한 테두리 광택.
*   **인터랙션**: 마우스 오버 시 블록 강조, 클릭 시 상세 내용 편집창 팝업.

## 5. 구현 로드맵
1.  **환경 설정**: Electron + Vite + React 프로젝트 초기화.
2.  **위젯 기본 창 구현**: 투명 배경, 프레임 제거, 항상 위(Always on Top) 설정.
3.  **그리드 시스템 구축**: 24시간을 5분 단위로 렌더링하는 컴포넌트 개발.
4.  **상태 관리 로직**: 블록 선택 및 텍스트 입력 기능 구현.
5.  **디자인 폴리싱**: Vanilla CSS를 통한 최종 스타일링 및 애니메이션 추가.
